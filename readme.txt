Feature-rich wishlist plugin with AJAX functionality, guest wishlists, and social sharing for WooCommerce stores.

== Description ==

Advanced WooCommerce Wishlist is a comprehensive wishlist solution that enhances your WooCommerce store with powerful wishlist functionality. This plugin provides a seamless shopping experience for your customers with features designed to increase engagement and sales.

= Key Features =

* **AJAX Wishlist Management** - Add/remove products instantly without page reload
* **Guest Wishlist Support** - Allow non-registered users to create wishlists
* **Multiple Wishlists** - Users can create and manage multiple wishlists
* **Social Sharing** - Share wishlists on social media platforms
* **Price Drop Notifications** - Get notified when wishlist items go on sale
* **Flexible Button Placement** - Customize wishlist button positions
* **Email Notifications** - Stay updated with wishlist activities
* **Responsive Design** - Works perfectly on all devices
* **WooCommerce Compatible** - Fully compatible with WooCommerce 3.0+

= Button Placement Options =

* After Add to Cart button
* Before Add to Cart button
* On product image (overlay)
* Custom shortcode placement

= Shortcodes =

* `[aww_wishlist]` - Display user's wishlist

== Installation ==

1. Upload the `advanced-wc-wishlist` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Ensure WooCommerce is installed and activated
4. Configure the plugin settings in WooCommerce > Settings > Wishlist
5. The wishlist page will be automatically created at `/wishlist/`

= Requirements =

* WordPress 5.0 or higher
* WooCommerce 3.0 or higher
* PHP 7.4 or higher

== Frequently Asked Questions ==

= Can guests create wishlists? =

Yes! The plugin supports guest wishlists. Guests can add products to their wishlist and the data is stored in their browser session.

= How many wishlists can a user create? =

By default, users can create up to 10 wishlists. This can be configured in the plugin settings.

= Can I customize the wishlist button appearance? =

Yes! You can customize button text, colors, and position through the plugin settings.

= Does the plugin work with all themes? =

The plugin is designed to work with most WooCommerce-compatible themes. If you experience issues, you can use the shortcode `[aww_wishlist_button]` to place the button manually.

= Can I export wishlist data? =

Yes, the plugin includes export functionality for wishlist data in the admin area.

= Are price drop notifications automatic? =

Yes, the plugin automatically checks for price drops and sends notifications based on your configured settings.

= Can I disable guest wishlists? =

Yes, you can disable guest wishlist functionality in the plugin settings.

= Is the plugin translation-ready? =

Yes, the plugin is fully translation-ready and includes .pot files for translation.
